🔧 Connect KDF Maintenance Database

Uses Google Firebase Realtime Database — separate from the Gate Log.

1
Create a free Firebase project

Go to console.firebase.google.com → Add project → name it kdf-maintenance → disable Analytics → Create.

2
Enable Realtime Database

Sidebar → Build → Realtime Database → Create database → any region → Start in test mode → Enable.

3
Register Web App & paste credentials

Project Overview → Add app → Web → any nickname → Register. Copy the firebaseConfig values below.

Please fill all four fields.

Credentials saved in this browser only.
